# 📚 Nossa Biblioteca Comunitária de Códigos

Bem-vindo(a)! Este é um espaço aberto para guardarmos scripts, trechos de código e ferramentas úteis. Aqui, todos podem visualizar, copiar e contribuir com seus próprios códigos.

---

## 📥 Como baixar ou copiar um código
Se você encontrou um código que deseja usar na sua máquina, siga estes passos:

1. **Para Copiar (Copy & Paste):**
   - Clique no nome do arquivo que você deseja abrir.
   - No canto superior direito do quadro de código, clique no ícone de **dois quadradinhos** (ícone de copiar).
   - O código será copiado para sua área de transferência. Basta colar no seu editor (VS Code, Notepad, etc).

2. **Para Visualizar (Raw):**
   - Clique no botão **"Raw"** acima do código para ver o texto puro, sem a interface do GitHub.

3. **Para Baixar o arquivo individual:**
   - Ao lado do ícone de copiar, existe uma **setinha para baixo** (Download raw file). Clique nela para salvar o arquivo direto no seu computador.

---

## 📤 Como colocar o SEU código aqui
Você pode adicionar arquivos diretamente pelo navegador, sem precisar instalar nada:

1. No topo da lista de arquivos, clique em **"Add file"** e depois em **"Create new file"**.
2. **Organização por pasta:** No campo "Name your file", digite seu nome, uma barra e o nome do arquivo.
   - Exemplo: `seu-nome/meu_script.py`
   - *Isso criará automaticamente uma pasta com o seu nome, mantendo tudo organizado.*
3. Cole o seu código no campo de texto principal.
4. Role a página até o final e clique no botão verde **"Commit changes"**.
